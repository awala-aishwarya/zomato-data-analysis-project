# Zomato Data Analysis Project

## Project Type
EDA + Classification

## Contribution
Individual

## Project Overview
This project analyzes Zomato restaurant reviews and metadata to understand
customer behavior, preferences, and factors influencing restaurant ratings.

The project includes:
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization (UBM approach)
- Hypothesis Testing
- Text Preprocessing (NLP)
- Feature Engineering
- Machine Learning Models
- Model Evaluation
- Deployment Readiness

## Machine Learning Models Used
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor
